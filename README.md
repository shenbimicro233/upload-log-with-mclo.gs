# upload-log-with-mclo.gs
基于mclo.gs的API的MC日志上传前端，使用ElementPlus作为UI框架
# 使用
- 创建一个VUE项目
  `npm init vue@latest`
- 安装ElementPlus
  `npm install element-plus --save`
- 将仓库中的 `app.vue` 覆盖到项目中 `src/app.vue` 文件
- 构建项目，获得静态资源文件 `npm run build`
